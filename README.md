# 🐍 Python Projects Collection

A collection of beginner-to-intermediate Python projects created to improve programming, problem-solving, and software development skills. These projects cover topics such as file handling, object-oriented programming, randomization, user input validation, encryption, and game development.

## 📂 Projects Included

### ⏰ Alarm Clock
A simple alarm clock that allows users to set an alarm for a specific time.

**Features**
- Set custom alarm time
- Audio notification
- Easy-to-use interface

---

### 🎯 Mastermind Game
A console-based implementation of the classic Mastermind game where players guess a secret color code.

**Features**
- Random code generation
- Hint system
- Multiple attempts
- Score tracking
- Color guide
- Replay option

---

### ➗ Math Test
A math quiz application that generates random arithmetic questions.

**Features**
- Random questions
- Score system
- Multiple difficulty levels
- Performance summary

---

### 🔢 Number Guessing Game
A game where the player guesses a randomly generated number.

**Features**
- Multiple difficulty levels
- Hint system
- Attempt counter
- Replay functionality

---

### 🔐 Password Generator
Generate strong and secure passwords with customizable options.

**Features**
- Adjustable password length
- Letters, numbers, and symbols
- Strong password generation
- Easy-to-use interface

---

### 🛡️ Password Manager
A secure password manager for storing login credentials.

**Features**

- Auto-lock protection

### ❓ Quiz Application
A multiple-choice quiz program.

**Features**
- Multiple questions
- Score calculation
- Instant feedback
- Final results

---

### ✊✋✌ Rock-Paper-Scissors Game
A classic Rock-Paper-Scissors game against the computer.

**Features**

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/python-projects.git
```

### Navigate to the Project

```bash
cd python-projects
```

### Run Any Project

```bash
python filename.py
```

Replace `filename.py` with the project you want to run.

Example:

```bash
python mastermind.py
```
